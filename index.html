Make.com AI-Powered Smart Gmail Attachment Processor

This Make.com scenario is built to intelligently process incoming Gmail attachments. It uses the OpenAI API (ChatGPT) to review the file content (PDFs, DOCX, XLSX, CSV), generate a short, descriptive new filename, and then rename and upload the file to Google Drive. A comprehensive log of the processing, including timestamps and both filenames, is maintained in Google Sheets, with an optional summary email sent after each file is handled.
